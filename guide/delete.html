刪除 Delete

物件資料刪除

格式：$objext->delete();。
PHP
結果
  // 藉由 id 等於 1
  $user = User::find('one', array(
      'must' => array('id' => 1)
    ));

  // 刪除
  $user->delete();

多筆刪除

格式：Class::deleteMany($ids);。
  • $ids - 陣列，primary_key 欄位的陣列。
PHP
結果
  $ids = array(1, 2, 3);

  // 依據 primary_key 欄位值陣列刪除
  User::deleteMany ($ids);
