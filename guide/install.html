Mac 上安裝 ElasticaSearch

安裝步驟

  • 先安裝 Java SDK。
  • Brew 安裝：brew install elasticsearch。
  • 設定指向：ln -sfv /usr/local/opt/elasticsearch/*.plist ~/Library/LaunchAgents。
  • 開啟 Elastica Searchlaunchctl load ~/Library/LaunchAgents/homebrew.mxcl.elasticsearch.plist。
  • 關閉 Elastica Searchlaunchctl unload ~/Library/LaunchAgents/homebrew.mxcl.elasticsearch.plist。

改使用 elasticsearch-rtf

  • 移至資料夾：cd /usr/local/Cellar/elasticsearch。
  • 複製下載 elasticsearch-rtf：git clone https://github.com/medcl/elasticsearch-rtf 1.0.0。
  • 切換 elasticsearch-rtf：brew switch elasticsearch 1.0.0。

安裝 plugin _head

  • 移至資料夾：cd /usr/local/var/lib/elasticsearch/plugins/。
  • 安裝 Plugin：plugin -install mobz/elasticsearch-head。
  • 使用瀏覽器開啟：http://127.0.0.1:9200/_plugin/head/。
